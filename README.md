# 📘 Code Alpha Student Grade

A simple Java project to manage student records and calculate grades.  
This project demonstrates object-oriented programming concepts such as classes, objects, and encapsulation.

---

## 🚀 Features
- Add and manage student details
- Console-based user interface
- Grade calculation and display
- Organized code structure with `Student`, `StudentManager`, and `ConsoleUI` classes

---

## 📂 Project Structure
src/
├── Main.java                # Entry point of the application
└── Students/
├── ConsoleUI.java      # Handles user interaction via console
├── Student.java        # Represents a student entity
└── StudentManager.java # Manages student records and operations


---

## 🛠️ Technologies Used
- Java (Core)
- Object-Oriented Programming (OOP)

---

## ▶️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Ishantsingh2812/CodeAlpha_StudentGrade.git

2. Navigate to the project folder
    cd CodeAlpha_StudentGrade

3. Compile the code
  javac src/Main.java

4.Run the program
